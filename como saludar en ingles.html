COMO SALUDAR EN INGLES

en ingles tenemos distintas formas de saludar y las podemos dividir en los de acuerdo al grado de formalidad ,tenemos las formales e informales

formales

>1.tenemos la opcion de decir "HELLO" que se traduce al espanol como "HOLA"

example:

  • HELLO maria!
  • HOLA maria!

2.podemos decir "GOOD MORNING" que se traduce como "BUENOS DIAS"

example:

  • GOOD MORNING mom!
  • BUENOS DIAS mamá!


informales

1.una buena opcion para saludar a tus cercanos es "HI!" que se puede traducir como "HOLA" pero de forma informal

example:

  • HI everyone!
  • HOLA a todos!

2."MORNING" al igual que "GOOD MORNING" se utiliza para dar los buenos dias, pero esta es su version informal

3.la expresion "what´s upp? se utiliza tanto para saludar como para saber como esta la persona

example:

  • what´s upp bury?
  • que tal amigo?
